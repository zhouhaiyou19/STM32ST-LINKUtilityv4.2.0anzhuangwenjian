## 使用说明
1. **下载文件**: 从本仓库下载STM32 ST-LINK Utility v4.2.0的安装文件。
2. **安装软件**: 双击安装文件，按照提示完成软件的安装过程。
3. **连接设备**: 使用ST-LINK调试器连接STM32系列MCU。
